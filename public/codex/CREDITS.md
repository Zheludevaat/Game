# Codex — Sources & Translations

Every entry in `src/game/data/codex.ts` quotes or paraphrases a
**public-domain** translation of a primary Hermetic, Neoplatonic, or
alchemical text. The `source` field on each `CodexEntry` names the
specific work + translator.

## Translations used (all public domain)

| Translator      | Work                                          | Date     |
|-----------------|-----------------------------------------------|----------|
| G. R. S. Mead   | *Thrice-Greatest Hermes* (Corpus Hermeticum)  | 1906     |
| Stephen MacKenna| *The Enneads* (Plotinus)                      | 1917–30  |
| Thomas Taylor   | *De Mysteriis Aegyptiorum* (Iamblichus)       | 1821     |
| Isaac Newton    | English rendering of the *Tabula Smaragdina*  | c. 1680  |
| Benjamin Jowett | *The Dialogues of Plato*                      | 1871     |

All five translators died before 1956; works are firmly public domain
in every jurisdiction that follows the Berne 70-year rule. Each entry
attributes its source so curious players can verify and read further.

## Note on Open5e / SRD content

The plan considered seeding new entries with Open5e SRD (D&D 5e
content under CC-BY-SA). On inspection, SRD spell / monster prose was
incompatible with the codex's Hermetic register — it would have read
as fantasy mechanics next to actual Plotinus. We instead added six
entries drawn from the same public-domain primary-source family that
already populates the codex (Mead, MacKenna, Taylor, Jowett, Newton).
No SRD content is bundled.
